# ⏰ Digital Clock

A simple web application that displays a real-time digital clock updated every second.

## 📋 Project Description

**Digital Clock** is a minimalist web application that displays the current time in 24-hour format (HH:MM:SS). This application features a modern design with a glassmorphism effect and a lime green color theme on a dark background.

## ✨ Main Features

- ⏱️ **Real-Time Clock** — Displays time updated every second
- 🎨 **Modern Design** — Interface with glassmorphism and glow effects
- 📱 **Responsive Design** — Accessible on various devices
- 🌑 **Dark Theme** — Eye-friendly dark mode
- 🎯 **Minimalist UI** — Focused on the clock display without unnecessary elements
